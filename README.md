# SQL-Czech-English-Calendar


CZ:
Tento script vytvoří tabulku 'calendar' která obsahuje sloupce s daty viz příklad:

date - 2023-01-01
date_cz - 01.01.2023
day - 1
month - 1
year - 2023
day_en - Sunday
day_cz - Neděle
month_name_en - January
month_name_cz - Leden

Datumy generuje od 1.1.2023 až do 31.12.2033
